# Vanessa Xavier Advocacia

Website profissional para o escritório de advocacia Vanessa Xavier, especializado em Direito Civil e Previdenciário.

## Tecnologias Utilizadas
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- GSAP (GreenSock Animation Platform) para animações de alta performance
- ScrollTrigger para efeitos de rolagem fluida
- Google Fonts (Playfair Display, Bodoni Moda, Inter)

## Estrutura do Projeto
- `index.html`: Estrutura principal do site
- `assets/`: Imagens, logos e favicons
- `css/style.css`: Estilização completa e design system
- `js/main.js`: Lógica de animações e interatividade

- [x] Design Responsivo (Desktop & Mobile)
- [x] Animações Premium com GSAP
- [x] Contato direto via WhatsApp, Instagram e E-mail
